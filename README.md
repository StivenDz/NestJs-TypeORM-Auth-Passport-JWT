# Dz Shop API
1. Clonar proyecto
2. ```pnpm install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar Base de datos (Postgres)

```
docker-compose up -d
```
6. Ejecutar SEED: 
```GET http://localhost:3000/api/seed```

7. Levantar: ```npm run start:dev```
